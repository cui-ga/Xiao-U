  小U，你身边贴心的智能医疗健康顾问。基于医疗知识图谱、自然语言处理与检索增强生成（RAG）技术，精准读懂你的健康问询，输出专业、规范、易懂的医疗健康信息参考。

## 一、项目背景
  日常健康问询场景中，用户普遍面临诸多痛点：网络健康信息鱼龙混杂、真伪难辨，非专业用户难以筛选权威内容；专业医疗术语门槛高，普通人难以理解晦涩的医学说明；身体出现轻微不适时，想提前了解基础病因、就医方向，却没有便捷可靠的渠道；多轮健康咨询时信息断层，无法获得连贯、递进的专业解答。

本项目专为解决上述问题打造，以严谨的医疗知识图谱为核心底座，结合业界领先的NLP与大模型RAG技术，构建了一套端到端的医疗智能问答系统。既保证了医疗信息的专业性与权威性，又兼顾了普通用户的使用体验，同时严格恪守「信息参考不替代专业诊疗」的医疗合规底线。

## 二、核心功能
1. 全场景医疗问答覆盖：支持疾病症状、治疗方案、用药指导、科室推荐、预防护理、病因分析等10+类医疗健康场景问询，完美适配口语化表达
2. 精准自然语言理解：精准识别用户问询意图与医疗实体，支持别名、简称、口语化表述的兼容处理，测试集意图识别准确率超98%
3. 双引擎知识驱动：融合结构化医疗知识图谱与权威医疗文献，既保证核心医疗信息的精准可溯源，又覆盖泛健康场景的丰富知识需求
4. 流畅多轮对话交互：自动关联上下文会话历史，支持连续追问、话题延续，还原自然的问诊咨询体验，无需重复补充信息
5. 极速响应体验：全链路多级性能优化，系统平均响应时间低于200ms，无需等待即可获得解答。rag生成时间由模型参数及配置要求决定。
6. 合规安全保障：所有回答均标注医疗信息参考提示，明确区分科普参考与专业诊疗，坚守医疗合规红线

## 四、系统整体架构
本系统采用模块化、流水线式的高可扩展架构，核心问答全链路流程如下：
<img width="693" height="117" alt="image" src="https://github.com/user-attachments/assets/a46eac68-df23-441b-9489-46034e7006c0" />


### 4.1 核心模块拆解
1. NLP自然语言理解核心：负责读懂用户的问询，完成文本清洗、意图分类、医疗实体识别与知识图谱链接，是整个系统的「理解中枢」
2. KG知识图谱查询引擎：基于用户意图与识别的医疗实体，从结构化医疗知识图谱中精准查询权威信息，是系统的「知识底座」
3. RAG检索增强生成模块：当图谱无匹配信息时，从海量医疗文献中检索相关内容，完成重排序与知识融合，是系统的「知识扩展中枢」
4. 对话交互与Web服务：负责多轮对话状态管理、用户交互界面渲染，提供简洁易用的操作体验，是系统与用户的「交互窗口」

## 五、核心技术实现
### 5.1 高精度NLP理解模块
- 意图识别：基于哈工大讯飞联合实验室开源的hfl/chinese-macbert-base预训练模型，针对医疗场景完成微调，适配10+类医疗问询意图分类，测试集准确率超98%
- 实体识别：采用工业界主流的混合策略方案，融合AC自动机万级实体词典高速匹配、正则规则适配医疗专属表达、Jieba分词优化实体边界，精准覆盖疾病、药品、科室、检查等核心医疗实体
- 实体链接与规范化：设计三层递进匹配策略（精确匹配→基于编辑距离的模糊匹配→同义词/缩写映射），将口语化、非标准化的实体表述，精准锚定到知识图谱的标准节点，核心实体识别与链接F1值超95%

### 5.2 高性能知识图谱查询引擎
- 首创模板化查询架构，将42种复杂医疗场景的Cypher查询，抽象为可配置的意图模板，新增问答需求无需修改代码，仅需更新配置模板即可完成扩展，大幅提升开发效率与系统可维护性
- 多级性能优化：内置LRU内存缓存、Neo4j连接池管理、查询失败自动模糊降级机制，保障服务99.9%可用性，平均查询响应时间低于200ms
- 智能结果格式化：将图谱返回的结构化键值对，自动转化为通顺易懂的自然语言回答，过滤无效空值，适配不同场景的表达习惯

### 5.3 增强型RAG检索生成系统
- 采用业界最佳实践的两阶段检索架构：先用M3E中文嵌入模型+FAISS向量检索，完成海量医疗文献的高召回；再用BGE交叉编码器对候选结果做精细化重排序，兼顾检索效率与准确率
- 多源知识智能融合：智能判断知识图谱与RAG检索结果的优先级，医疗核心信息优先以结构化图谱权威内容为主，泛健康科普场景以文献丰富内容为补充，生成的答案兼具准确性与全面性
- 生产级工程优化：实现模型单例缓存、查询结果LRU缓存、大模型生成重试与超时控制，全链路异常处理，保障系统稳定高效运行

### 5.4 流畅的多轮对话系统
- 完整的对话状态管理，自动留存与关联会话历史，支持连续追问、话题跳转，无需用户重复补充信息
- 基于Gradio构建的简洁易用Web界面，支持对话内容一键复制、新对话开启、历史清空等便捷功能
- 全链路日志与用户反馈收集机制，支撑模型持续迭代优化，不断提升问答效果
